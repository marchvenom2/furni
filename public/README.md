# Furniture Con Furry Site - Static Website

A responsive, SEO-friendly static furniture website built with HTML5, CSS3, and JavaScript.

## 📋 Features

- **Responsive Design**: Mobile-first approach, fully responsive on all devices
- **SEO Optimized**: Meta tags, structured data, semantic HTML, XML sitemap
- **Google Ads Ready**: Pre-configured ad spaces for AdSense monetization
- **Multiple Pages**: Home, About Us, Sitemap, Privacy Policy
- **Working Images**: All images use placeholder service (no broken images)
- **Accessibility**: WCAG compliant with keyboard navigation
- **Performance**: Lightweight, fast-loading static pages
- **Modern Styling**: Attractive gradient design with smooth animations

## 📁 File Structure

```
furniture-con-furry-site/
```

## 🖼️ Working Images

All product and background images use **placeholder.com** service:
- Logo: `https://via.placeholder.com/50x50?text=FCF`
- Product Images: `https://via.placeholder.com/400x300?text=ProductName`
- Hero Background: `https://via.placeholder.com/1920x400?text=Premium+Furniture`

These placeholders work perfectly and will display until you replace them with your actual images.

## 🚀 Getting Started

1. Download all files
2. Create a folder named `furniture-con-furry-site`
3. Save all HTML, CSS, and XML files into this folder
4. Open `index.html` in your browser to preview
5. Update Google AdSense publisher ID in HTML files
6. Replace placeholder images with your own furniture photos
7. Update contact information with your details
8. Upload to your hosting provider
9. Submit sitemap to Google Search Console

## 📊 Analytics Integration

Add Google Analytics by including this before closing `</head>`:
```html
<script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'GA_MEASUREMENT_ID');
</script>
```
